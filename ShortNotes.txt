List ->

Array List -> 			implements list interface
			  	allows null value
			  	maintain order
				
LinkedList -> 			similar to array list
			  	it implements double link list
			  	allows null value
vector ->	  		Is thread safe
			  	synchronized
			  
Set -> 

HashSet ->    			implements set interface
			  	allows null value
			  	does not maintain insertion order
			  	does not allows duplicate character

LinedHashSet ->			Similar to set
				Maintain insertion order -- IMP
				Allows null value
				
TreeSet -> 			does not allow duplicate value
				does not allow null value -- IMP
				access and retrieve quite fast as it implements navigable set
				Tree Set maintains accesnding order. 
				first() method to get first data
				last() method will get the last data
				
Map ->				

HashMap ->			Implements MAP interface
				Stores value in key value pair format
				no duplicate key
				allows one null key
				no insertion order
				
LinkedHashMap ->		Similar to map. 
				Maintain insertion order
				
TreeMap ->			similar to map
				maintain ascending order.
